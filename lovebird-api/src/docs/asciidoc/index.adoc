[[overview]]
= Overview

[[overview_http_verbs]]
== HTTP Method

|===
| Verb | Usage

| `GET`
| Used to retrieve a resource

| `POST`
| Used to create a new resource

| `PUT`
| Used to create a new resource or replace data representing the target resource

| `DELETE`
| Used to delete an existing resource
|===

[[overview_http_status_codes]]
== Http Status Code

|===
| Status code | Usage

| `200 SUCCESS(OK)`
| The request completed successfully

| `400 FAIL(BAD REQUEST)`
| The request was failed.

| `500 ERROR(INTERNAL SERVER ERROR)`
| The server encountered an unexpected condition which prevented it from fulfilling the request.
|===

[[overview_Lovebird_response_rule]]
== Lovebird Response Rule

|===
| code | message

| `1000`
| 요청에 성공하셨습니다.

| `1100`
| Bearer 접두사가 포함되지 않았습니다.

| `1101`
| 잘못된 jwt 토큰입니다.

| `1200`
| 잘못된 인증 제공자 입니다..

| `1201`
| 중복된 회원가입 입니다.

| `1202`
| 존재하지 않는 회원 입니다.

| `1250`
| 존재하지 않는 프로필입니다.

| `1300`
| 이미 커플 연동을 한 유저입니다.

| `1301`
| 자기 자신과는 커플 연동을 할 수 없습니다.

| `1400`
| 유효하지 않은 사용자 정보입니다..

| `8000`
| 잘못된 파라미터 입니다.

| `9998`
| 내부 서버 에러 입니다.

| `9999`
| 외부 서버 에러 입니다.
|===
